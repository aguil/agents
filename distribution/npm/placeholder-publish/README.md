# `@aguil/agents` bootstrap (0.0.0)

This directory exists **only for the very first
[`npm publish`](https://docs.npmjs.com/cli/v11/commands/npm-publish)** so the
**`@aguil/agents`** name exists on the registry before you configure **Trusted
Publishing** and ship real builds from **`.github/workflows/release.yml`**.

Replace this tarball with **`0.1.0+`** via CI by pushing a semver tag
(**`v0.1.0`**, …).

Do **not** depend on this version in production.
